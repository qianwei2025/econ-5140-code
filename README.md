# ECON 5140: Applied Econometrics

This repository contains code demonstrations and examples for ECON 5140: Applied Econometrics at Northeastern University.

## Contents

### Lesson 1: Generalized Linear Models (GLM)
- `lesson1_glm_demo.ipynb` - Comprehensive demonstration of GLM methods using tech company examples
  - Binary outcomes: Logistic and Probit regression (ad click prediction)
  - Count outcomes: Poisson regression (API usage prediction)
  - Model comparisons and interpretations

## Requirements

- Python 3.7+
- numpy
- pandas
- matplotlib
- seaborn
- scipy
- statsmodels

Install dependencies with:
```bash
pip install numpy pandas matplotlib seaborn scipy statsmodels
```
